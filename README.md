# 📊 Stock Price Analysis

## 📌 Project Overview

This project analyzes historical stock data of major tech companies (AAPL, AMZN, GOOG, MSFT).
The goal is to identify trends, relationships, and patterns using data analysis and visualization.

---

## 🎯 Objectives

* Understand stock price trends over time
* Compare multiple companies
* Analyze correlations between stocks
* Study daily returns (percentage change)

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset

Historical stock price data including:

* Open, Close, High, Low prices
* Date
* Volume

---

## 📈 Analysis & Visualizations

### 🔹 1. Stock Price Trends

* Plotted stock prices for AAPL, AMZN, GOOG, MSFT
* Applied moving averages (10, 20, 50 days)

👉 Insight:

* All stocks show an **overall upward trend**
* Moving averages help identify **trend direction and smoothing**

---

### 🔹 2. Moving Average Analysis

* Compared short-term vs long-term trends

👉 Insight:

* When short-term average crosses above long-term → **uptrend signal**
* When it crosses below → **downtrend signal**

---

### 🔹 3. Correlation Analysis (Pairplot)

👉 Insight:

* Strong positive correlation between tech stocks
* Stocks often move in similar directions
* Useful for portfolio diversification analysis

---

### 🔹 4. Daily Returns Analysis

👉 Insight:

* Most returns are centered around 0 → normal market behavior
* Some extreme spikes → high volatility days
* MSFT shows relatively stable returns compared to others

---

## 📊 Key Insights

* Tech stocks are highly correlated
* AMZN and GOOG show strong growth over time
* MSFT is more stable with fewer extreme fluctuations
* Moving averages clearly highlight trend patterns

---

## 📌 Conclusion

This project demonstrates how stock data can be analyzed to understand:

* Market trends
* Risk (volatility)
* Relationships between stocks

Such analysis helps in making better financial and investment decisions.
